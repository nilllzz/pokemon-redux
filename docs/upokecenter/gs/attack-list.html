This is a complete description of the moves in the Gold and Silver games, and their effects. Extra explanation can be found in the supplement to this list.

This move list also applies to the other generation 2 games Pokémon Crystal and Stadium 2.

Flag g - An accuracy check is performed for this attack. Thus, Brightpowder, if held by the opposing Pokémon, can decrease this attack's accuracy.

Flag f - If the user is holding King's Rock, there is a 30/256 chance that the opposing Pokémon will flinch if the opposing Pokémon lost HP because of this attack and the user struck first with this attack. (Prevented by Substitute, but not if the substitute just faded.)
